# RustCoreutils #
Lite implementation of GNU core utilities in Rust   
**Only for GNU/Linux**
![gnu/linux](/theme/linux-svgrepo-com.svg)
## Installation ##
`git clone https://github.com/Crazyninja59/rust_coreutils.git`
## Usage ##
```
cd rust_coreutils/
cargo run [utility] [arguments]
```
